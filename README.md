# AppleJuice

Este projeto é uma aplicação para o M5StickCPlus2 que manda pacotes de Bluetooth para despositivos apple 

## Requisitos

- M5StickCPlus2
- Arduino IDE
- Bibliotecas necessárias:
  - M5StickCPlus2.h
  - BLEDevice.h
  - BLEUtils.h
  - BLEServer.h
